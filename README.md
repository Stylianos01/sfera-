# Digital Sphere Experiment

A frontend visualization project focusing on advanced CSS techniques.

## Features
- **3D Depth:** Achieved using multiple radial gradients.
- **SVG Filters:** Integrated fractal noise (`feTurbulence`) for the grain texture.
- **Performance:** No image files used; purely code-based rendering.
